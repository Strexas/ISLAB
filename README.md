# INSTALLATION
## Clone the project
```commandline
git clone git@github.com:Strexas/ISLAB.git
```
## Create Virtual Environment
```commandline
cd ISLAB
python3 -m venv .venv
```
## Activate VENV
For Linux
```commandline
source .\.venv\Scripts\activate
```
For Windows
```commandline
source .venv/bin/activate
```
## Install libraries
```commandline
pip install -r requirements.txt
```

## Run the project
```commandline
python3 app.py
```
